## Admin Console Login

The EPIC Admin Console provides secure access for authorized administrators and club managers through email and password authentication.

### Login Process

Access to the admin console is restricted to pre-approved accounts only. Follow these steps to sign in:

1. **Navigate to Admin Console**: Go to the admin console login page
2. **Enter Credentials**: Use your registered email address and password
3. **Sign In**: Click the "Sign In" button to access your dashboard

### Important Notes

<Warning>
**Account Activation Required**: All admin console accounts must be activated by a system administrator before they can be used to log in.
</Warning>

<Note>
**No Registration Available**: Unlike the mobile app, the admin console does not allow self-registration. All accounts must be created and activated by existing administrators.
</Note>

### Login Features

- **Remember Me**: Option to stay logged in across browser sessions
- **Forgot Password**: Password reset functionality for existing accounts
- **Secure Authentication**: Industry-standard security protocols
- **Session Management**: Automatic logout for security

### Getting Access

If you need access to the admin console:

1. **Contact Administrator**: Reach out to your club's system administrator
2. **Account Creation**: Administrator will create your account with appropriate permissions
3. **Account Activation**: Administrator will activate your account
4. **Credentials Provided**: You'll receive your login credentials
5. **First Login**: Use provided credentials to access the admin console

### Troubleshooting

<AccordionGroup>
  <Accordion title="Can't log in with correct credentials">
    Ensure your account has been activated by an administrator. Inactive accounts cannot log in even with correct credentials.
  </Accordion>

  <Accordion title="Forgot password">
    Use the "Forgot password?" link on the login page to reset your password. This only works for activated accounts.
  </Accordion>

  <Accordion title="Need admin access">
    Contact your club administrator or system administrator to request account creation and activation.
  </Accordion>
</AccordionGroup>

### Security Best Practices

- Use a strong, unique password for your admin account
- Enable "Remember Me" only on trusted devices
- Log out when finished, especially on shared computers
- Report any suspicious login activity immediately
- Keep your login credentials confidential
