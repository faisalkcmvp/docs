## Phone Number Login

Use your registered phone number to access your EPIC Padel account with SMS verification.

### Step-by-Step Instructions

1. **Open the EPIC Mobile App**
   - Launch the app on your device
   - Tap "Login" on the welcome screen

2. **Select Phone Login**
   - Choose "Login with Phone" option
   - You'll see the phone number input form

3. **Enter Your Phone Number**
   - Select your country code from the dropdown
   - Enter your phone number (without country code)
   - Ensure the number is correct before proceeding

4. **Request Verification Code**
   - Tap "Send Code" button
   - A 6-digit SMS code will be sent to your phone
   - This may take 1-2 minutes to arrive

5. **Enter Verification Code**
   - Check your SMS messages for the verification code
   - Enter the 6-digit code in the app
   - Tap "Verify" to complete login

### SMS Code Issues

**Didn't receive the code?**
- Check your SMS/messages app
- Ensure you have cellular signal or WiFi
- Wait 2-3 minutes before requesting a new code
- Tap "Resend Code" if needed

**Code expired or invalid?**
- Request a new verification code
- Enter the code quickly after receiving it
- Codes typically expire after 10 minutes

### Troubleshooting

**Phone Number Not Recognized?**
- Verify you're using the correct country code
- Ensure the phone number matches your registered number
- Remove any spaces, dashes, or special characters
- Contact support if you've changed your phone number

**SMS Not Arriving?**
- Check if SMS blocking is enabled on your device
- Ensure you have sufficient cellular signal
- Try switching between WiFi and cellular data
- Contact your mobile carrier if issues persist

<Warning>
Make sure you're using the same phone number you registered with your EPIC Padel account.
</Warning>

### International Numbers

- Select the correct country code from the dropdown
- Enter only the local phone number (without country code)
- International SMS may take longer to arrive
- Standard SMS rates may apply depending on your carrier

### Security Features

- SMS codes expire after 10 minutes for security
- Each code can only be used once
- Multiple failed attempts may temporarily block login
- Your phone number is kept secure and private
